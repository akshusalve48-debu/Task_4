# Call Center Sales Performance Dashboard
# Project Overview
This project presents an interactive Call Center Performance Dashboard built using Power BI.
The dashboard helps analyze agent performance, call efficiency, and sales conversion to support business decision-making.
# Objective
Monitor total calls handled
Measure pickup rate and conversion rate
Evaluate agent productivity
Analyze call duration impact on sales
# Dataset Columns
AgentID – Unique agent ID
Agent_Name – Name of agent
CallID – Unique call record
CustomerID – Customer identifier
PickedUp – 1 = Yes, 0 = No
Duration – Call duration (seconds)
ProductSold – 1 = Yes, 0 = No
# Key KPIs
Total Calls
Pickup Rate (%)
Conversion Rate (%)
Average Call Duration
Total Sales
# Dashboard Features
KPI summary cards
Agent-wise performance comparison
Sales vs Duration analysis
Interactive slicers and filters
Multi-page navigation
# Tools Used
Power BI
DAX
Microsoft PowerPoint
# Outcome
This dashboard helps identify top-performing agents, improve conversion rates, and support data-driven decisions.
